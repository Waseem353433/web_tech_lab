# Personal Website - HTML Lab Project

This project is a simple multi-page website created as part of an HTML lab assignment.

## Features
- Personal Profile Page
- Weekly Course Schedule Table
- Cross-page Navigation
- Active Page Highlighting
- Back to Home Links
- Custom 404 Error Page

## Pages Included
- index.html (Home Page)
- profile.html (Profile Page)
- schedule.html (Course Schedule)
- contact.html (Contact Page)

## Technologies Used
- HTML5
- Basic CSS
- GitHub Pages
